# ktg-mes

### 系统简介
苦糖果MES系统是一款B/S结构、开源、免费的生产执行管理系统。

此系统基于本人多年离散智造行业的业务经验及J2EE项目经验开发。

主要目的是为国内离散制造业的中小企业提供一个专业化、通用性、低成本的MES系统解决方案。

系统将提供“售前”、“实施”、“用户培训”、“运维”等多个阶段的指导文档和教学视频，以帮助非IT专业人士顺利完成MES项目的实施应用。
